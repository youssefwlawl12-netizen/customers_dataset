# Online Retail Sales Dashboard 📊

## Overview
This project is an interactive Power BI dashboard built using the Online Retail dataset. It provides insights into sales performance, customer behavior, product performance, and order analysis to support business decision-making.

## Dashboard Features

### Executive KPIs
- Total Revenue
- Total Orders
- Total Customers
- Total Quantity Sold
- Average Order Value
- Number of Countries

### Sales Analysis
- Monthly Revenue Trend
- Monthly Orders Trend
- Revenue by Country
- Top Products by Revenue
- Top Products by Quantity Sold

### Customer Analysis
- Top 10 Customers by Revenue
- Customer Segmentation (VIP, Gold, Silver, Regular)
- Customer Distribution by Segment

### Order Analysis
- Number of Orders
- Average Items per Order
- Average Revenue per Order

### Interactive Features
- Date Filter
- Country Filter
- Product Filter
- Customer Filter
- Drill-through pages for Country Details
- Drill-through pages for Customer Details

## Tools Used
- Microsoft Power BI
- DAX
- Power Query

## Dataset
**Online Retail Dataset**

The dataset contains transactional sales data including:
- Invoice Number
- Product Description
- Quantity
- Unit Price
- Customer ID
- Country
- Invoice Date

## DAX Measures
Some measures used in this project include:

- Total Revenue
- Number of Orders
- Total Customers
- Average Order Value
- Average Items per Order
- Customer Segmentation

## Project Objective

The goal of this dashboard is to provide business insights by answering questions such as:

- How much revenue has the business generated?
- Which products perform the best?
- Who are the highest-value customers?
- Which countries generate the most revenue?
- What is the average value of an order?
- How are customers segmented based on spending?
